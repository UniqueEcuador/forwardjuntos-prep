# 🧠 Je Brein van Binnen — MS & Dendriet Onderzoek

Een interactieve, toegankelijke webpagina die recent wetenschappelijk onderzoek naar dendritische voltage-imaging uitlegt voor mensen met MS en andere chronische hersenaandoeningen.

## ✨ Wat zit erin

| Bestand | Omschrijving |
|---|---|
| `index.html` | Hoofdpagina met alle secties |
| `css/style.css` | Volledige styling — warm, organisch, toegankelijk |
| `js/canvas-bg.js` | Geanimeerde neuron-achtergrond (hero) |
| `js/simulator.js` | Interactieve dendriet-simulator met MS-modus |
| `js/glossary.js` | Uitklapbare woordenlijst (12 termen) |
| `js/main.js` | Scroll-animaties & algemene interacties |

## 🚀 Gebruik

### Lokaal draaien
Open `index.html` direct in je browser — geen build-stap nodig.

### Op GitHub Pages zetten
1. Push de map naar een GitHub-repository
2. Ga naar **Settings → Pages**
3. Kies **Branch: main**, map **/ (root)**
4. Sla op — je site is binnen minuten live op `https://jouwgebruikersnaam.github.io/jouwrepo`

## 🎛️ Interactieve features

- **Neuron-simulator**: Stuur zelf een signaal door een virtuele hersencel
- **MS-schakelaar**: Zie hoe MS-schade de signaaloverdracht verstoort
- **Burst vs. enkel signaal**: Vergelijk de twee signaaltypes
- **Sterktemeter**: Visuele weergave van signaalverlies met afstand
- **Woordenlijst**: 12 moeilijke termen simpel uitgelegd

## 🎨 Design

- Lettertypen: DM Serif Display + DM Sans (Google Fonts)
- Kleurpalet: Warm crème · Oranje accent · Donkere simulator
- Volledig responsive (mobiel & desktop)
- Toegankelijk: semantische HTML, toetsenbordnavigatie, contrastrijke kleuren

## 📖 Wetenschappelijke basis

Gebaseerd op gepubliceerd onderzoek naar in vivo dendritische voltage-imaging in CA1 pyramidale neuronen, met gelijktijdige meting van voltage- en calcium-signalen op meerdere dendritische locaties.

**Drie kernbevindingen:**
1. Bursts propageren betrouwbaarder terug dan losse signalen
2. Somato-dendritische koppeling neemt af met afstand van soma
3. Elektro-calcium koppeling neemt af met toenemende takorde
